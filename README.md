
# [Arpit Yadav](https://arpityadav.me)
 
Website made using Nuxt 3. Design insipired by [Brittany Chiang](https://https://brittanychiang.com/.com/)

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

Feel free to use this however you like.
